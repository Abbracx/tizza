# tizza
### Tizza is mini application that will demonstrate the use of monolith and microservices.

## Technologies

1. Docker
2. RabbitMQ or Kafka
3. Redis
4. celery

## Demonstrated concepts
1. Crosss service communications 
    - RPC 
    - Messaging (pub/subs and queues)
2. Sharding
3. Rate Limiting
4. Bearer token authentication
5. Testing
